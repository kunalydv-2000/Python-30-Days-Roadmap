# Task Tracker CLI

A beginner-friendly Python mini project built during the final stretch of the 30-day roadmap.

## Goal
Build a command-line app that can create, list, complete, and delete tasks while storing data in JSON.

## Core Features
- add a task
- list all tasks
- mark a task as complete
- delete a task
- save tasks to a local JSON file
- load tasks when the program starts

## Stretch Features
- due dates
- priorities
- filtering
- search
- colored terminal output
